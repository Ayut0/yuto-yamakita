## 自己PR

- 行動力
  カナダ・バンクーバーでの3年間、現地のカレッジに通いながらコネゼロの状態から就職活動を行い、北米現地採用を実現。カナダでの就業経験を通じて培った英語力は、英語圏チームとの即戦力としての連携に活かせるレベルに達している。

- フルスタック×フィンテックの生産実績
  UPSIDERではフロントエンド開発を主導しながら、バックエンド（Go・Kotlin）・インフラ・セキュリティ領域にも越境して貢献。CIパイプラインを最大54%短縮し、チーム全体のフィードバックループを加速。また、週3回発生していた早朝の手動リカバリー作業（約2時間/回）をバッチ処理の安定化により撲滅した。PCI-DSS 4.0.1監査でゼロ指摘という結果に貢献するなど、信頼性と開発速度の両立を実践してきた。

- 勉強会、ミートアップへの積極的な参加
  ネットワーク構築、知識拡充の為、オンライン・オフライン問わず定期的に勉強会やミートアップへの参加。

- コミュニケーション能力・英語
  カレッジ在学中から多国籍チームのリーダーを担い、現職でもSecurity Championとして社内セキュリティ意識の啓発を主導。40名規模の全社オフサイトをエンド・ツー・エンドで企画・運営するなど、エンジニアリング以外の組織貢献にも積極的に関わる。

- 拡張性を意識した設計思想
  機能開発においては常に変更容易性・シンプルさを軸に設計し、単なる動作するコードではなく「将来のチームメンバーが迷わず触れるコード」を書くことを意識している。O'Reilly『Software Architecture: The Hard Parts』のブッククラブ参加など、継続的なインプットも欠かさない。
